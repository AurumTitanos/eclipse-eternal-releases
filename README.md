# Eclipse Eternal — Releases

Download builds for invited testers.

Builds are **password-protected** — testers, use the password you were given (ask in the tester chat if you don't have it). After unzipping, open **READ ME FIRST.txt**.

## First run
Windows may show "Windows protected your PC" (the build isn't code-signed). It's safe — click
**More info → Run anyway** (one-time per version). If your antivirus quarantines it, allow the folder.
