## Is it safe? (Why Windows warns about it)
**Yes.** On first run Windows may show **"Windows protected your PC"** (Microsoft Defender SmartScreen), and some antivirus may be cautious about it. This is **expected for a brand-new build from a small, independent project — it is not a malware detection.**

SmartScreen is **reputation-based**: it warns about any app that isn't code-signed and hasn't yet been downloaded by lots of people, because it has *no established reputation yet* — not because anything was found wrong with the file. As more testers run it, that reputation builds and the warning goes away. Code-signing certificates (which build that trust faster) cost money and require identity verification, so an early alpha skips them for now. **The warning is about signing/reputation, not safety.**

- **Microsoft's own explanation** of how SmartScreen reputation works — note the line *"When a file is not signed, SmartScreen reputation must build for each new version of your files, starting with zero reputation"*: <https://learn.microsoft.com/en-us/windows/apps/package-and-deploy/smartscreen-reputation>
- **Third-party coverage** of SmartScreen warning hard about unsigned apps: <https://www.theregister.com/2020/06/05/windows_10_microsoft_defender_smartscreen/>
